JavaScript Array Exercises

1. Array Basics

  1. Create an array of 5 colors and print the second and last color.
  2. Create an array of numbers and replace the first element with 100.
  3. Create an array with mixed data types (string, number, boolean). Print the type of each element using a loop.
  4. Create a nested array and print the inner arrayâ€™s second element.
  5. Create an array using the Array constructor and display its length.

2. push(), pop(), shift(), unshift()

  1. Create an array of fruits and add one fruit at the end using push().
  2. Remove the last element from the array using pop() and display the updated array.
  3. Add two new fruits at the beginning using unshift().
  4. Remove the first element using shift() and print the new array.
  5. Use a combination of push() and pop() to keep only 3 elements in your array.

3. slice() and splice()

  1. Create an array [10, 20, 30, 40, 50] and use slice() to extract [20, 30].
  2. Using the same array, use splice() to replace 30 with 35.
  3. Insert 25 before 30 using splice().
  4. Remove the last two elements using splice().
  5. Use slice() to copy the first three elements into a new array.

4. concat(), join(), toString()

  1. Create two arrays [1,2,3] and [4,5,6] and merge them using concat().
  2. Convert an array of names into a string separated by commas using join().
  3. Use toString() on an array and display the output.
  4. Merge three arrays using concat() and find the length of the new array.
  5. Create an array of city names and join them using ' - ' as a separator.

5. Search Methods (indexOf, lastIndexOf, includes)

  1. Create an array ['apple', 'banana', 'orange', 'banana', 'grape'] and find the first index of 'banana'.
  2. Find the last index of 'banana' using lastIndexOf().
  3. Check if 'mango' exists in the array using includes().
  4. Write code to check if a specific number exists in an array, if not, push it.
  5. Use includes() to verify if 'kiwi' is present in your fruit array and print a message accordingly.

6. find() and findIndex()

  1. Create an array [5, 12, 8, 130, 44] and find the first number greater than 10.
  2. Use findIndex() to locate the position of the first element greater than 50.
  3. Find the first even number in an array using find().
  4. Find the index of the first odd number in an array using findIndex().
  5. Use find() to search for a string longer than 5 characters in an array of words.

7. sort(), reverse(), copyWithin(), flat()

  1. Sort an array [5,3,8,1,2] in ascending order.
  2. Reverse the sorted array and print it.
  3. Use copyWithin() to copy first two elements to the end of an array [10,20,30,40,50].
  4. Flatten a nested array ['a','b',['c','d'],['e',['f']]] using flat().
  5. Use flat() with depth 2 to flatten ['x',['y',['z',['w']]]].

8. length, Destructuring & Misc

  1. Create an array of 10 numbers and print its length.
  2. Remove the last element using pop() and print the new length.
  3. Use array destructuring to extract first three elements into variables.
  4. Create a constant array and modify one element (to show arrays are mutable).
  5. Combine multiple array methods (push, splice, sort, reverse) to manipulate data dynamically.
