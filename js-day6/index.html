JavaScript Strings & Methods

A String is a sequence of characters enclosed in quotes.

String Declaration

let name1 = "Sathish"; // Double quotes
let name2 = 'Developer'; // Single quotes
let name3 = `JavaScript`; // Template literal

String Methods

1. Changing Case

  • .toUpperCase()
  • .toLowerCase()
let text = "hello";
console.log(text.toUpperCase()); // HELLO
console.log(text.toLowerCase()); // hello

2. Access Characters

  • str[index]
  • .charAt(index)
let str = "Sathish";
console.log(str[0]); // S
console.log(str.charAt(3)); // h

3. Search Methods

  • .indexOf()
  • .lastIndexOf()
  • .includes()
let text = "I love JavaScript";
console.log(text.indexOf("love")); // 2
console.log(text.includes("Java")); // true

4. Extracting Strings

  • .slice(start, end)
  • .substring(start, end)
  • .substr(start, length) (old)
let str = "JavaScript";
console.log(str.slice(0,4)); // Java

5. Replace

  • .replace()
  • .replaceAll()
let text = "JS is awesome. JS is fun.";
console.log(text.replace("JS","JavaScript"));
console.log(text.replaceAll("JS","JavaScript"));

6. Trim Spaces

  • .trim()
  • .trimStart()
  • .trimEnd()
let msg = "   Hello JS   ";
console.log(msg.trim()); // "Hello JS"

7. Split String

  • .split()
let data = "html,css,js";
console.log(data.split(",")); // ['html','css','js']
