Destructuring allows you to unpack values from arrays into variables.

üîπ Example:

// Output: red green

5Ô∏è‚É£ All JavaScript Operators

JavaScript has different types of operators to perform actions on data.

üîπ Types of Operators:

  • Arithmetic: +, -, *, /, %, ++, --
  • Assignment: =, +=, -=, *=, /=, %=
  • Comparison: ==, ===, !=, !==, >, <, >=, <=
  • Logical: &&, ||, !
  • Bitwise: &, |, ^, ~, <<, >>
  • Ternary: condition ? true : false
  • Type: typeof, instanceof
  • Spread/Destructure: ... (spread), [ ] / { } (destructure)

üîπ Example:
