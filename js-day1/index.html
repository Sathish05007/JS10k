JavaScript Basics

Introduction to JavaScript

What is JavaScript?

JavaScript is a lightweight, interpreted programming language used to make web pages interactive and dynamic. It was created by Brendan Eich in 1995 and is one of the core technologies of the web.

History of JavaScript

JavaScript was created by Brendan Eich in 1995 for Netscape Navigator. Initially, it had different names during development:

  • Mocha -Original internal code name
  • LiveScript -Renamed before release
  • JavaScript -Final name for marketing reasons

Features

  • Lightweight and Fast
  • Object-Oriented
  • Interpreted Language
  • Platform Independent
  • Event-Driven

Applications of JavaScript

  • Web Development
  • Mobile App Development (React Native)
  • Game Development
  • Server-side Programming (Node.js)

JavaScript Engines

JavaScript engines execute JS code in different browsers or platforms:

  • V8 -Google Chrome & Node.js
  • SpiderMonkey -Mozilla Firefox
  • Chakra -Microsoft Edge (Legacy)
  • JavaScriptCore -Safari

Example code works in all engines:

function greet(name) {
  return "Hello, " + name + "!";
}
console.log(greet("Sathish"));

Output Statements

console.log() -Shows message in browser console.

alert() -Displays alert box.

prompt() -Takes user input.

console.log("Hello World");
alert("Good Morning");
prompt("Enter your name:");

Including JavaScript in HTML

There are three ways to include JS in your web page:

  • Inline JS: Inside an HTML tag.
  • Internal JS: Inside <script> tag within the page.
  • External JS: Linked via <script src="index.js">.

Internal JavaScript Example

<script>
  console.log("Hello from Internal JS!");
  alert("Welcome to JavaScript!");
  let name = prompt("Enter your name:");
  console.log("User name:", name);
</script>

External JavaScript Example

External JS is written in a separate file and linked with <script src="index.js">.

<script src="index.js"></script>
