JavaScript Loops

Loops help to repeat a block of code multiple times.

1️⃣ for Loop

Most common loop with initialization, condition & increment.

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

2️⃣ while Loop

Runs as long as the condition is true.

let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}

3️⃣ do...while Loop

Executes at least once even if condition is false.

let n = 1;
do {
  console.log(n);
  n++;
} while (n <= 5);

4️⃣ for...of Loop

Used to iterate arrays / strings.

let arr = ["HTML", "CSS", "JS"];
for (let item of arr) {
  console.log(item);
}

5️⃣ for...in Loop

Used to iterate object keys.

let user = {name: "Sathish", age: 22};

for (let key in user) {
  console.log(key, ":", user[key]);
}

6️⃣ break Statement

Stops the loop immediately.

for (let i = 1; i <= 5; i++) {
  if (i === 3) break;
  console.log(i);
}
// Output: 1, 2

7️⃣ continue Statement

Skips the current iteration.

for (let i = 1; i <= 5; i++) {
  if (i === 3) continue;
  console.log(i);
}
// Output: 1, 2, 4, 5
