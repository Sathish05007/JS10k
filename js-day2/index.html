üîπ Data Types

JavaScript has two main types:

  • Primitive: number, string, boolean, null, undefined, bigint, symbol
  • Non-primitive (Reference): object, array, function, date, etc.

üîπ Primitive Data Types Examples

  • Number: let num = 100;
  • String: let name = "Sathish";
  • Boolean: let isActive = true;
  • Null: let empty = null;
  • Undefined: let value;
  • BigInt: let big = 12345678901234567890n;
  • Symbol: let sym = Symbol("id");

üîπ Non-Primitive Data Types

Non-primitive types include objects and arrays:

let arr = [1, 2, 3, 4];
let obj = {name: "Sathish", age: 25};
console.log(arr[0]); // 1
console.log(obj.name); // "Sathish"
