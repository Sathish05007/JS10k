JavaScript If, Else, Else If & Switch

Conditional statements are used to make decisions in JavaScript based on conditions.

1. if Statement

Executes a block of code if a condition is true.

let age = 18;

if (age >= 18) {
  console.log("You are eligible to vote");
}

2. if...else Statement

Runs one block if condition is true otherwise runs another.

let marks = 45;

if (marks >= 35) {
  console.log("Pass");
} else {
  console.log("Fail");
}

3. else if Statement

Used to check multiple conditions in sequence.

let score = 85;

if (score >= 90) {
  console.log("A Grade");
} else if (score >= 75) {
  console.log("B Grade");
} else if (score >= 50) {
  console.log("C Grade");
} else {
  console.log("Fail");
}

4. Nested if

if statement inside another if

let user = "admin";
let isLogged = true;

if (isLogged) {
  if (user === "admin") {
    console.log("Welcome Admin");
  } else {
    console.log("Welcome User");
  }
}

5. Ternary Operator (Short If-Else)

Short form of if else.

let age = 20;
let message = age >= 18 ? "Adult" : "Minor";
console.log(message);

6. switch Statement

Used when multiple conditions are checked on same variable.

let day = 3;

switch(day) {
  case 1: console.log("Monday"); break;
  case 2: console.log("Tuesday"); break;
  case 3: console.log("Wednesday"); break;
  default: console.log("Invalid Day");
}
